# rkivacc

Downloads and parses the latest Vaccination Data for Germany from the Robert Koch Institut.

**This library is not made by, endorsed by or in any way related to the RKI.**

## Usage

See [example.py](example.py).

## Requirements

* At least Python3
* request
* openpyxl
